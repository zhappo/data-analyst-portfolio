# Análise de Vendas Superstore
Projeto da Semana 1 analisando vendas no dataset Superstore, focando em categorias líderes na região Oeste e sua evolução anual (2014-2017). Usa Python com Pandas, Matplotlib e Seaborn para EDA completa.
​

# Instalação
Clone o repositório.

Instale dependências: pip install pandas matplotlib seaborn.

Execute semana_1.ipynb no Jupyter/Colab.

# Uso
Abra semana_1.ipynb.

Rode as células sequencialmente.

Dataset carrega automaticamente do GitHub: https://raw.githubusercontent.com/yajasarora/Superstore-Sales-Analysis-with-Tableau/master/Superstore%20sales%20dataset.csv (9.994 linhas, 21 colunas).
​

# Funcionalidades
Carregamento e limpeza de dados.

Análise exploratória (shape, head, duplicatas).

Filtros por região Oeste e ano.

Gráficos de vendas por categoria/subcategoria e evolução temporal.

Conclusões sobre líderes de vendas (ex: Furniture e Technology).
​
